## Lancer le programme en local

Télécharger le projet
```sh
git clone https://github.com/jibenight/jquery-project
```
Installer les gems
```sh
bundle install
```
Genérer la base de donnée
```sh
rails db:create
```
Lancer le serveur
```sh
rails server
```
Lien vers le localhost :  [http://localhost:3000/ ](http://localhost:3000/ )
